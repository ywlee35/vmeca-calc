하중 케이스별 흡착력 공식

Case I: 수평 흡착 – 수직 이동만

F = (W × (g + a) × S) / n
g=9.81 m/s², a=가속도, S=안전계수, n=컵수

Case II: 수평 흡착 – 수직 + 수평 이동

F = S × √[(W×a/μ)² + (W×g)²] / n
μ=마찰계수

Case III: 수직 흡착 – 수직 방향 이동

F = S × √[(W×g/μ)² + (W×a)²] / n
